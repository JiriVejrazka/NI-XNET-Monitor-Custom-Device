NI-XNET Monitor Custom Device

General description

This is an Hardware Inline Asynchronous Custom Device which allows you to open subordinate session(s) to an existing NI-XNET ports and allows you to monitor port condition.

Supported Interfaces

NI-XNET Monitor currently supports CAN, FlexRay and LIN.
